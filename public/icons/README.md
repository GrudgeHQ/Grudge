# PWA Icons Directory

This directory contains Progressive Web App icons in various sizes for different devices and platforms.

## Required Icon Sizes:
- 72x72 - Android density-independent pixels (mdpi)
- 96x96 - Android density-independent pixels (hdpi)
- 128x128 - Chrome Web Store
- 144x144 - Android density-independent pixels (xhdpi)
- 152x152 - iPad touch icon
- 192x192 - Android density-independent pixels (xxhdpi)
- 384x384 - Android splash screen
- 512x512 - Android density-independent pixels (xxxhdpi)

## Icon Design Guidelines:
- Use the app's primary theme color (#7c3aed - purple)
- Include the "G" logo or sports equipment iconography
- Ensure icons work well on both light and dark backgrounds
- Use maskable icons that work with various device icon shapes
